# School Login System

## Description
A simple front-end school login system created using HTML and CSS.  
This project demonstrates basic page navigation and form usage.

## Pages Included
- register.html – User registration page
- login.html – Login page
- dashboard.html – Dashboard page
- styles.css – Styling for all pages

## How to Use
1. Open register.html
2. Fill in the registration form and click Register
3. You will be redirected to the Login page
4. Enter username and password
5. Click Login to access the Dashboard
6. Click Logout to return to Login page

## Note
This project is front-end only and does not store user data.
